Este proyecto consiste en el desarrollo de un sistema de monitoreo inteligente para licuadoras, basado en sensores de vibración y movimiento, cuyo objetivo es detectar a tiempo el desgaste de las aspas, el desbalanceo o el sobreesfuerzo del motor. A través de un dispositivo IoT —integrado por un microcontrolador y sensores como acelerómetros o giroscopios— se recopilan datos en tiempo real sobre el comportamiento del equipo.

La información se procesa y se envía a una interfaz interactiva donde el usuario puede visualizar el estado de la licuadora, recibir alertas tempranas de fallas, y obtener recomendaciones de mantenimiento para prevenir accidentes, mejorar el rendimiento del aparato y prolongar su vida útil.

El proyecto está dirigido tanto a usuarios domésticos como a cocineros de restaurantes o cafeterías, ofreciendo una herramienta intuitiva, accesible y universal que permite tomar decisiones informadas sobre el uso seguro y eficiente de la licuadora.
